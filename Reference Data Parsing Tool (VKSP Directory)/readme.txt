Project:
Reference Data Parsing Tool (VKSP Directory)

Problem:
Organizations regularly receive large reference datasets (e.g., enforcement service directories), which require integration into internal databases.
Raw data formats are not optimized for direct analytical use.

Solution:

Developed parsing scripts to transform and load directory data into a database
Implemented logic for adapting data structures to target systems
Created additional scripts to standardize regional data formats

Tech:
SQL Server (v11.0), SSMS 2014

Impact:

Simplified integration of external reference data
Improved data consistency and usability for analytics
Reduced manual preprocessing effort
