Project:
Debt Recovery Target Identification System

Problem:
Outstanding debts (both judicial and non-judicial) may remain unresolved due to lack of structured identification processes.
Manual detection of such cases is inefficient and error-prone.

Solution:

Developed SQL logic to identify debtors eligible for write-off or further processing
Designed adaptable query structure that can be calibrated for different CRM systems
Ensured anonymization for portability across environments

Impact:

Automated identification of actionable debt cases
Reduced manual workload in debt analysis
Improved efficiency of debt recovery decision-making
