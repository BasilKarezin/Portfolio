🧾 Project: Bankruptcy Status Monitoring System
📌 Description

This project is a Python-based data solution designed to monitor, validate, and update bankruptcy statuses using external data sources.

The system automates the process of identifying individuals and entities, retrieving relevant information, and ensuring that internal database records remain accurate and up-to-date.

⚙️ Key Features
Automated search and matching of individuals based on:
Tax Identification Number (INN)
Full name (FIO)
Insurance number (SNILS)
Integration with external data sources and API responses
Parsing and structuring of JSON responses
Matching external data with internal database records
Automated validation and updating of bankruptcy status
End-to-end data pipeline from data retrieval to database update
📈 Impact
Improved data accuracy in financial records
Reduced risk of incorrect financial decisions
Increased automation of manual verification processes
Enhanced reliability of internal monitoring systems
🧠 Technologies Used
Python
SQL
JSON processing
Data parsing and validation techniques
🔍 Project Structure

The project consists of multiple interconnected components handling:

Data retrieval and search logic
Data parsing and normalization
Matching and validation logic
Database update and synchronization
💡 Key Learning
Designing end-to-end data pipelines
Handling and validating real-world data
Working with external data sources and inconsistencies
Building reliable data processing workflows
