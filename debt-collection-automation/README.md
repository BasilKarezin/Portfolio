# Debt Collection Workflow and Legal Cost Automation

## 📌 Overview
This project focuses on automating debt collection processes, including legal workflows and cost calculations.

The system was designed to handle large portfolios of debt cases and reduce manual processing in legal operations.

---

## ⚠️ Problem
Debt collection is a multi-stage legal process:

1. Court order is issued  
2. Debtor may cancel it  
3. Additional legal actions are required  

Manual processing leads to:
- Errors in calculations (state duties, claims)
- Slow decision-making
- High operational workload

---

## ⚙️ Solution
Developed a SQL-based system that:

- Extracts full debt portfolio data  
- Automates calculation of state duties  
- Embeds reusable formulas  
- Supports bulk processing  

---

## 💰 Impact
- Reduced manual workload  
- Improved accuracy  
- Faster decision-making  

---

## 🛠 Tech Stack
- SQL (MS SQL Server)
