Project:
PDF Merging Automation Tool

Problem:
Combining multiple PDF documents into structured packages is often required, but highly variable depending on context.
Static scripts lack flexibility for real-world use cases.

Solution:

Developed a flexible Python-based framework for merging PDF files
Designed the script to be easily customizable for different document structures
Provided a reusable base for building tailored document processing workflows

Impact:

Simplified document preparation processes
Reduced manual effort in handling large volumes of files
Enabled scalable and adaptable document automation
